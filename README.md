# Investment Summaries

### Prerequisites

This app requires Node.js and npm (which should come with Node, but check just in case)

### Install

Server:

```sh
cd server
npm i
```

Client:

```sh
cd client
npm i
```

### Run the app

To run locally:

```sh
cd server
node app.js
cd ../client
npm start
```
